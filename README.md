## Description du projet :
Le projet consiste en une animalerie contenant plusieurs boutiquues. Chaque boutique dispose d'une adresse, de plusieurs produits disponibles, et d'animaux. Les animaux sont de différentes espèces possibles (dans l'exemple, seulement chat et poisson).

## Contenu du projet :
### - 'src/main/java/org.example/entities'
Contient les classes représentant les différentes entités du projet : Animal, Boutique, Produit, Adresse.

### - 'src/main/java/org.example/enumeration'
Contient les énumérations utilisées dans le projet, avec FishLivEnv, ProdType.

### - 'src/main/java/resources/META-INF/persistence.xml'
Fichier de configuration pour la persistance des données, spécifiant les paramètres de connexion à la base de données et les classes d'entités.

### - 'pom.xml'
Fichier de configuration Maven, spécifiant les dépendances nécessaires pour le projet, telles que Hibernate
